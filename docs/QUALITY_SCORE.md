# QUALITY_SCORE.md — 품질 기준

## 코드 품질 체크리스트

새 코드 병합 전 확인:

- [ ] `npm run build` 오류 없음
- [ ] TypeScript 타입 오류 없음 (`npx tsc --noEmit`)
- [ ] `"use client"` 누락 없음
- [ ] API 키 클라이언트 노출 없음
- [ ] props 인터페이스 정의됨
- [ ] 에러 케이스 처리됨 (try-catch 또는 null 체크)
