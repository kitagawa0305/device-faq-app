# 기술 부채 트래커

## 현재 부채

| 항목 | 영향도 | 설명 |
|---|---|---|
| 입력 길이 제한 없음 | 중 | 사용자 질문에 최대 길이 제한 미적용 |
| 진단 결과 만료 없음 | 하 | `diag_` localStorage 캐시에 TTL 미설정 |
| 모바일 레이아웃 미지원 | 중 | 현재 데스크탑 고정 레이아웃 |
| 에러 로깅 없음 | 중 | API 실패 시 콘솔 외 기록 없음 |

## 해결 완료

| 항목 | 해결일 |
|---|---|
| API 키 클라이언트 노출 | 최초 설계 시 방지 |
| 무한 API 호출 | 6턴 제한 + 캐시로 해결 |
