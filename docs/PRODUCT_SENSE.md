# PRODUCT_SENSE.md — 제품 방향성

## 핵심 가치

**"기기 문제를 스스로 해결할 수 있게 돕는다"**

단순 FAQ 답변이 아니라, 사용자가 문제를 진단하고 수리 여부를 직접 판단할 수 있도록 충분한 정보를 제공합니다.

## 타겟 사용자

- 기기 고장 시 서비스센터 방문 전 스스로 진단하고 싶은 일반 사용자
- 수리 비용이 합리적인지 판단하고 싶은 사용자
- 간단한 수리는 직접 해보려는 DIY 사용자

## 핵심 차별점

1. **부품별 인터랙티브 도해** — 어디가 문제인지 시각적으로 확인
2. **AI 동적 진단** — 증상에 따라 맞춤형 질문으로 정확한 진단
3. **수리 비용 투명성** — DIY 여부 + 예상 비용으로 의사결정 지원
4. **사진 분석** — 육안으로 확인 어려운 손상도 AI가 분석

## 절대 하지 않을 것

- 수리를 강요하거나 특정 서비스센터를 추천하지 않음
- 부정확한 진단으로 사용자를 오도하지 않음
- 개인 정보(기기 모델, 증상 등)를 서버에 저장하지 않음
